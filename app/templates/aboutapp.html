About this app

This app is developed for work and educational purposes, it offers a tool for calculate mechanical stress in busbar arrangement on swithcgears and controlgrears based on IEC standars. If you want to know more about this project you can check it on GitHub (link below) and share suggestions. This project will be changing over time adding new features and technologies, also will be deploying on several web services like render, google cloud (App Engine), GitHub pages and others. Please feel free to share your issues sending a email to joosorio@utp.edu.co
